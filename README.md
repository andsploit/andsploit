andsploit

======

andsploit 是 android渗透测试平台

andsploit(10/10/2013)从drozer移植过来,旨在提供方便快捷的android平台生头测试。

andsploit 让你可以快速的搜索发现app应用,设备中的漏洞,并且提供方便快捷的后渗透攻击模块。从而对他们进行后渗透攻击
